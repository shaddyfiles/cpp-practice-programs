# C++ Basics

This repository contains beginner-level C++ programs created while learning
core programming concepts such as variables, conditions, loops, functions,
and basic logic building.

## Topics Covered
- Input / Output
- If-Else statements
- Loops (for, while, do-while)
- Functions
- Simple programs and logic exercises

## Purpose
This repository is intended for learning and practice.
